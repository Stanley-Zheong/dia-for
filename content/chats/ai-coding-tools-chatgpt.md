## User

我想比较 Cursor、Claude Code 和传统 IDE 插件，应该从哪些角度看？

## ChatGPT

可以从三个层面比较：

1. **上下文能力**：是否理解整个项目结构。
2. **执行能力**：是否能直接修改文件、运行测试、修复错误。
3. **协作成本**：是否能把你的意图转成稳定的工程改动。

如果你主要做产品原型和全栈项目，Cursor 会更像一个带 AI 的工作台。

## User

那公开写一篇文章的时候，应该怎么表达差异？

## ChatGPT

建议不要只列功能表，而是用具体任务来比较。例如：

> 给三个工具同一个重构任务，看它们如何读代码、提出方案、落地修改、验证结果。

这种方式更接近真实使用体验。
